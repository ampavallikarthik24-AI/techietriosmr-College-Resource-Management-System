Project: College Resource Management - TechieTrios MR
DB: techietriosmr
How to run:
1. Start XAMPP (Apache + MySQL).
2. Place this folder in C:\xampp\htdocs\techietriosmr
3. Open http://localhost/techietriosmr/
4. Use UI to view/add/edit/delete records.
Tables included: department, faculty, course, room, equipment, timetable, allocation_log
